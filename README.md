# 사용법

다음 명령어로 소스 코드를 가져와 주세요.

`git clone https://github.com/KYankee6/naver-land-real-pyungso.git`

이후 확장 프로그램 (주소창 chrome://extensions/)에 들어가, 해당 폴더를 업로드할 수 있습니다.


"압축해제된 확장 프로그램을 로드합니다." 버튼을 누르면 사용 가능합니다!
